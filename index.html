Quick Start

Installation

npm install -g zombie-port-killer

Or use yarn or pnpm:

yarn global add zombie-port-killer
pnpm add -g zombie-port-killer

Usage

Kill a process on a specific port:

zkill 3000

Kill without confirmation:

zkill 3000 --force

List all active ports:

zkill scan

Show system information:

zkill info
